선택자(Selector)란? ★★★

인라인 스타일 방식

1. 전체(모든 요소) 선택자

        [표현법]
        * {
            스타일 속성 : 스타일속성 값;
        }
        => 현재 이 html 문서 내의 모든 요소를 다 선택할 때 사용

2. 태그 선택자

        [표현법]
        태그명{
            스타일속성 : 스타일속성 값;
            스타일속성 : 스타일속성 값;
        }
        => 문서 내에 같은 태그들을 전부 선택할 떄 사용

3. 아이디 선택자(#아이디속성값)

        [표현법]
        #아이디속성값{스타일속성 : 스타일속성값;}

        => 이 문서 내에서 특정 HTML 요소 딱 하나만을 선택할 때 사용
        단, 해당요소에 반드시 ID속성을 이용해서 이 문서내에 고유한 아이디값이 부여되어 있어야한다
        동일한 태그더라도 부분적으로 다르게 스타일을 부여하고 싶을 때 사용

4. 클래스 선택자(.클래스 속성값)

        [표현법]
        .클래스속성값{
            스타일 속성 : 값;
        }
        => 현재 html문서에서 여러 개의 요소 선택할 때 사용
