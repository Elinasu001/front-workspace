    radio button 과 checkbox

    < action="checked">
    radio button (type="radio")

    - 여러 개의 선택지를 중에서 오로지 한 개만 선택하게 하고 싶을 때
    - name속성 값이 똑같은 것들끼리 하나의 그룹으로 묶임
    - 제출 시 현재 선택된(checked) value가 넘어감

    숙소 위치 평가 :
    9+ 우수
    8+ 좋음
    7+ 일반

    type="checkbox"

    - 여러 개의 선택지들 중에서 여러 개를 선택 가능하게 하고 싶을 때 씀
    - 넘기고자 하는 값을 value속성의 값으로 넣어주어야함
